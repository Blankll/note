# springboot

## 创建springboot应用





### 启动spring 项目

mvn spring-boot:run



## 注解

@controller: 处理http请求

@RestController: Spring4之后的注解，@ResponseBody + @Controller

@RequestMapping:  配置url映射

## url参数

@PathVariable 获取url中的数据　``/paht/1/name``

@RequestParam 获取请求参数  `` /path?id=1``

@GetMaping 组合注解
